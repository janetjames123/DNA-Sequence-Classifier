# 🧬 GeneInsight AI

An AI-powered bioinformatics tool that interprets genetic variants from VCF files and generates clinically meaningful explanations.

## 🚀 Features

* Parses VCF files
* Maps variants to genes
* Assigns impact scores
* Links variants to disease information
* Generates AI-based clinical explanations

## 🧠 Tech Stack

* Python
* cyvcf2
* pandas
* transformers (Hugging Face)
* Google Colab

## 📊 Pipeline

VCF → Variant → Gene → Impact → Disease → AI Explanation

## 📁 How to Run

1. Upload a VCF file in Google Colab
2. Run all cells
3. Get annotated variant output
